# My Next.js Boilerplate

This is a custom Next.js boilerplate with support for Tailwind CSS, TypeScript, Axios, and more!

## Features
- Next.js
- Tailwind CSS
- TypeScript or JavaScript support
- Prettier, ESLint, and Husky setup
